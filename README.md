## 🎁 Installation

```shell
npm i
```

## 🏃‍♂️ Run

Running webpack-dev-server, with `webpack mode=development`. HMR(Hot Module Replacement) supported.

```shell
npm run start
```

## 🏗 Build

Builds React project in `/build` directory.

```shell
npm run build
```
